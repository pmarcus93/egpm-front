**Demo:** https://egpm.netlify.app/

## Requirements

- Node.js >= 18.0.0

## Installation

Clone the repository and install dependencies:

```bash
npm install
```

## Development

Start the local development server with hot-reloading:

```bash
npm run dev
```

## Build

Generate production files:

```bash
npm run build
```

Preview the production build:

```bash
npm run serve
```
